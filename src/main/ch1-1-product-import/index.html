Product Import

First you need to import products to Persoo. Either you already have a XML product feed for price comparators (Heureka, Zbozi,cz) or for Google Merchant Center or Facebook advertisements.

After having products in Persoo, you can start using them in all offer types (recommendation boxes, emails, web widgets, ...). Persoo collects user interactions with products only for product IDs (we call them itemID) and needs to regularly read your product feed to learn about current price, title, link, image link,...

Instructions

  1. 1
    Go to Setting - imports and in the list select "products".
  2. 2
    Add url https://s3-eu-west-1.amazonaws.com/cz.persoo.images/sampleProductFeeds/snapbacksSample.json to import url and select JSON type
  3. 3
    Click on "Status" section in top bar and then click "run" import in test environment to import the products from the JSON.
  4. 4
    Click the status line to see details. Find "destinations" url to preview and download results.
  5. 5
    Note: destinations url contains secret hashes -- set your own in Settings - Credentials - secret hash.
  6. 6
    Publish changes to production.
  7. 7
    Run import in production (go to Import status, and click "run" in production environment.
