Installing persoo snippet

Install persoo snippet to page header.

Instructions

  1. 1
    Go to Settings - installation and copy javascript snippet to page header.
  2. 2
    Open chrome console. Type persoo to chech, that persoo object is loaded.
  3. 3
    Open persoo console by adding ?persooAdminBar=1 to url.
