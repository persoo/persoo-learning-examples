Getting started

Using Persoo is very easy. Persoo is loaded to your website as any other javascript. All you need is to include "Persoo loader" javascript into the header of each page of your website. Persoo loader is generated for your account during registration, or you can get it in Persoo Admin. Persoo loader will asynchronously load all resources it needs. Asynchronous loading does not block your page, and thus it does not increase the page loading time.

See this page source code for an example how to include Persoo loader in an HTML page.

After Persoo loader is included, you can call Persoo functions 

persoo('functionName',parameters);

Calling Persoo functions is safe - you never lose your commands. Before Persoo client is loaded, it stores your commands into a command queue. The client processes your command queue when loaded. After loading, your commands are processed immediately.

See Persoo API for the list of all supported functions.

When data are sent to Persoo using 'send' command, the server processes your request using a specified workflow and returns data plus possible triggers (javascript functions).

What does Persoo loader do?

Persoo loader asynchronously loads two resources:

  • persoo.js - Persoo javascript client, which
    • provides Persoo API, i.e., sending data to Persoo cloud
    • collects data from your website (using plugins or custom functions you defined in the admin)
    • triggers actions - receives decisions from Persoo and triggers functions defined in actions.js
  • actions.js - implementation of javascript functions which you can trigger as Persoo actions. There are both predefined functions and custom functions. You can add your own functions through Persoo Admin (you can register your own javascript functions using Persoo API).
